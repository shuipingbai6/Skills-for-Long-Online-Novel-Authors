# 网文作者蒸馏系统

将网文作者的创作风格、叙事技巧、世界观构建能力蒸馏成可复用的 AI Skill。

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 创建作者 Skill

```
/create-author
```

或直接说：
- "帮我创建一个作者 skill"
- "我想蒸馏一个作者"
- "新建作者"

### 3. 使用生成的 Skill

```
/{笔名} 帮我续写一段玄幻小说开头
```

## 目录结构

```
author-skill/
├── SKILL.md                    # 主入口文件
├── prompts/                    # Prompt 模板
├── tools/                      # 工具脚本
├── authors/                    # 【生成的 Skill 存放位置】
│   └── {slug}/
│       ├── SKILL.md           # 完整 Skill
│       ├── writing.md         # 写作能力
│       ├── author_persona.md  # 作者人格
│       └── meta.json          # 元数据
└── knowledge/                  # 【原始材料存放位置】
    ├── novels/                # 小说文件
    ├── comments/              # 评论数据
    └── social/                # 社交媒体内容
```

## 工作流程

```
触发 Skill → 基础信息录入 → 原材料导入 → 分析原材料 → 生成预览 → 写入文件 → 完成
```

### 详细步骤

| 步骤 | 说明 |
|------|------|
| 1. 触发 | 说 `/create-author` 或 "帮我创建一个作者 skill" |
| 2. 录入 | 回答 3 个问题：笔名、基本信息、性格画像 |
| 3. 导入 | 提供小说文件、评论数据、社交媒体内容 |
| 4. 分析 | 提取写作风格和作者人格 |
| 5. 预览 | 确认生成的 Skill 摘要 |
| 6. 写入 | 创建 `authors/{slug}/` 目录和文件 |
| 7. 完成 | 可通过 `/{slug}` 触发使用 |

## 数据收集

### 小说文件

| 格式 | 说明 | 存放位置 |
|------|------|----------|
| txt | UTF-8 编码，自动识别章节 | `knowledge/novels/` |
| epub | 标准 2.0/3.0 格式 | `knowledge/novels/` |

### 评论数据

| 平台 | 格式 | 存放位置 |
|------|------|----------|
| 起点 | 时间 + 用户 + 内容 | `knowledge/comments/` |
| 晋江 | [用户] 内容 | `knowledge/comments/` |
| 番茄 | 楼层 + 内容 | `knowledge/comments/` |

### 社交媒体

| 平台 | 格式 | 存放位置 |
|------|------|----------|
| 微博 | 时间 + 内容 | `knowledge/social/` |
| 公众号 | 标题 + 正文 | `knowledge/social/` |

## 标签体系

### 写作风格标签

| 标签 | 说明 |
|------|------|
| 水文大师 | 详细描写、慢节奏叙事 |
| 剧情紧凑 | 快节奏、情节密集 |
| 慢热型 | 前期铺垫长、后期高潮迭起 |
| 爽文流 | 快节奏打脸、金手指频繁 |
| 虐心流 | 喜欢给主角设置困境 |
| 种田流 | 细节描写丰富、建设过程详细 |
| 无敌流 | 主角开局就强、碾压式战斗 |
| 凡人流 | 主角资质平平、靠努力成长 |

### 更新习惯标签

| 标签 | 说明 |
|------|------|
| 日更党 | 每天稳定更新 |
| 周更党 | 每周固定时间更新 |
| 月更党 | 更新不稳定、单次字数多 |
| 太监王 | 经常断更、烂尾风险高 |
| 爆发帝 | 平时更新少，偶尔爆发 |
| 稳定型 | 字数和时间都很稳定 |

### 人格标签

| 标签 | 说明 |
|------|------|
| 毒舌作者 | 评论回复犀利、爱吐槽 |
| 宠粉狂魔 | 积极回复评论、重视读者反馈 |
| 高冷型 | 很少回复评论、保持神秘感 |
| 自黑型 | 经常自嘲、与读者互动轻松 |
| 段子手 | 评论区和作品里都是段子 |
| 认真帝 | 回复评论认真详细 |
| 傲娇型 | 嘴硬心软、反差萌 |
| 佛系作者 | 更新随缘、心态平和 |

### 题材标签

| 标签 | 说明 |
|------|------|
| 玄幻 | 世界观宏大、修炼体系 |
| 都市 | 现代背景、都市生活 |
| 科幻 | 未来设定、科技元素 |
| 历史 | 历史背景、考据严谨 |
