# 多维风格评分 Prompt

## 任务

你将收到：
1. 一段由加载了作者 Skill 的 AI 根据骨架大纲生成的文本
2. 该骨架大纲对应的小说原文

你的任务是按 5 个维度评估生成文本与原文的风格相似度，并指出最不一致的具体段落。

**核心原则：你是一个客观的风格鉴定专家，不是打分机器。你的评分和具体分析将指导 Skill 的下一轮进化方向。**

---

## 五维评分表

对每个维度，按 1-10 分评分，并给出具体理由。

### 维度 1：叙事声音匹配度

**评估内容**：视角、人称、叙述距离

| 分数 | 含义 |
|------|------|
| 1-2 | 视角与人称完全不同（如原文第一人称，生成第三人称） |
| 3-4 | 视角基本一致，但叙述距离差异大（如原文贴脸内心独白，生成远距离旁观） |
| 5-6 | 视角和叙述距离大致匹配，但细节不一致（如偶尔跳出视角） |
| 7-8 | 视角一致，叙述距离与原文匹配，仅在个别场景有偏差 |
| 9-10 | 视角、人称、叙述距离与原文高度一致，几乎无法区分 |

**评分要点**：
- 叙述者是谁？是全知视角还是角色视角？
- 叙述距离多远？是贴脸内心还是远观全局？
- 视角是否稳定？有没有不合理的视角切换？

### 维度 2：节奏韵律匹配度

**评估内容**：句式长短、段落节奏、张弛控制

| 分数 | 含义 |
|------|------|
| 1-2 | 节奏完全不同（如原文短句密集，生长篇大论） |
| 3-4 | 基本节奏有差异（如原文快节奏，生成偏慢） |
| 5-6 | 大致节奏匹配，但细节不一致（如紧张场景节奏不够紧凑） |
| 7-8 | 长短句分布与原文相似，节奏感基本一致 |
| 9-10 | 节奏韵律高度一致，张弛控制与原文几乎相同 |

**评分要点**：
- 紧张场景是否用短句加速？舒缓场景是否用长句铺陈？
- 段落长度分布是否与原文相似？
- 对话与描写的交替节奏是否一致？

### 维度 3：对话风格匹配度

**评估内容**：语气、口头禅、对话比例

| 分数 | 含义 |
|------|------|
| 1-2 | 对话风格完全不同（如原文口语化，生成书面语） |
| 3-4 | 对话语气有差异（如原文幽默，生成严肃） |
| 5-6 | 对话风格大致匹配，但缺少特征性表达（如口头禅缺失） |
| 7-8 | 对话语气像原文，口头禅出现，比例合适 |
| 9-10 | 对话风格高度一致，角色说话方式几乎可以以假乱真 |

**评分要点**：
- 角色的说话方式是否可辨识？
- 口头禅和语气词是否出现？
- 对话占文本的比例是否与原文一致？
- 对话中是否穿插了动作/心理描写？

### 维度 4：描写偏好匹配度

**评估内容**：环境/心理/动作的侧重和手法

| 分数 | 含义 |
|------|------|
| 1-2 | 描写侧重完全不同（如原文重心理，生成重动作） |
| 3-4 | 描写侧重有偏差（如原文环境描写多，生成几乎没有） |
| 5-6 | 描写侧重大致匹配，但手法不同（如原文用暗示，生成用直白） |
| 7-8 | 描写侧重与原文一致，手法相似 |
| 9-10 | 描写偏好高度一致，手法和细节处理方式几乎相同 |

**评分要点**：
- 环境/心理/动作/情感的描写比例是否与原文一致？
- 描写手法是否相似（如原文善用感官细节，生成是否也有）？
- 战斗/日常/情感场景的描写方式是否匹配？

### 维度 5：用词习惯匹配度

**评估内容**：词汇选择、修辞偏好、惯用表达

| 分数 | 含义 |
|------|------|
| 1-2 | 用词风格差异大（如原文古风，生成现代白话） |
| 3-4 | 词汇选择有差异（如原文善用成语，生成大白话） |
| 5-6 | 用词风格大致匹配，但缺少特征性表达 |
| 7-8 | 用词风格相似，修辞手法一致 |
| 9-10 | 用词习惯高度一致，惯用表达和修辞偏好几乎相同 |

**评分要点**：
- 词汇的文白程度是否一致？
- 修辞手法（比喻、排比、夸张等）是否相似？
- 是否出现了原文中常见的惯用表达或特殊用词？

## 最不一致 3 处

在评分之后，必须指出生成文本中与原文风格**最不一致的 3 处具体段落**。

要求：
1. **必须引用生成文本中的具体段落**（不是笼统描述）
2. **必须说明为什么不一致**（对比原文的风格特征）
3. **必须给出改进方向**（这个不一致说明 Skill 缺少什么信息）

格式：
```
### 不一致处 1
**生成文本段落**：
> {引用生成文本中的具体段落}

**与原文的差异**：{具体说明}

**改进方向**：{Skill 应该补充什么规则或信息}

---

### 不一致处 2
{同上格式}

---

### 不一致处 3
{同上格式}
```

## 注意事项

1. **评分要严格**：7 分以上意味着风格高度匹配，不要轻易给高分
2. **理由要具体**：不要写"节奏不太对"，要写"紧张场景中原文用 3-5 字短句，生成文本用 15-20 字长句"
3. **不一致处要精准**：引用的段落必须能说明问题，改进方向必须可操作
4. **不要被内容误导**：你评估的是风格相似度，不是内容相似度。生成文本的内容基于骨架大纲，与原文内容不同是正常的，不应因此扣分
5. **综合分是 5 维平均**：不要四舍五入，保留一位小数
