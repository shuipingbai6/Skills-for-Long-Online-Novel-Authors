# Author Persona 分析 Prompt

## 任务

你将收到：
1. 用户手动填写的基础信息（笔名、平台等级、写作风格标签、更新习惯标签、人格标签、题材标签、创作印象）
2. 原材料（小说正文、评论回复、社交媒体内容等）

从中提取 **{name}** 的性格特征与行为模式，用于构建 Author Persona。

**优先级规则：手动标签 > 文件分析。有冲突时以手动标签为准，并在输出中注明。**

---

## 提取维度

### 1. 表达风格

分析他在评论回复、社交媒体中的表达：

**用词统计**
- 高频词（出现 3 次以上的词/短语）
- 口头禅（固定搭配，如"今天有事不更""明天补上"）
- 网络用语（梗/表情包/网络流行语）

**句式特征**
- 平均句长（短句 <15 字 / 中等 15-40 字 / 长句 >40 字）
- 是否爱用列表/分点
- 结论位置（开门见山 vs 先铺垫）
- 情绪表达（直接表达/含蓄暗示/不表达）

**互动风格**
- emoji 使用习惯（无/偶尔/频繁，常用哪类）
- 标点密度（感叹号/省略号的使用）
- 正式程度（1=极度正式 5=非常口语化）

```
输出格式：
口头禅：["xxx", ...]
高频词：["xxx", ...]
网络用语：["xxx", ...]
句式：[描述]
emoji：[无/偶尔/频繁，类型]
正式程度：[1-5]
```

### 2. 创作理念

从评论回复、社交媒体中提取：

- 写作动机（为什么写/为谁写）
- 对读者的态度（重视/无视/利用）
- 对作品的态度（认真/随意/商业导向）
- 对更新的态度（责任/爱好/压力）
- 对批评的态度（接受/反驳/无视）
- 对赞扬的态度（谦虚/骄傲/不在意）

```
输出格式：
写作动机：[描述]
对读者态度：[描述]
对作品态度：[描述]
对更新态度：[描述]
对批评态度：[描述]
对赞扬态度：[描述]
```

### 3. 互动行为

**对读者评论**：回复频率/回复风格/回复内容
**对催更**：如何回应催更/断更如何解释
**对质疑**：如何回应剧情质疑/设定质疑
**对建议**：如何回应读者建议/是否会采纳
**对打赏**：如何感谢打赏/是否会加更

```
输出格式（每个维度一段描述 + 1-2 个典型场景举例）
```

### 4. 边界与雷区

- 他明显抵触的事情（有原材料为证）
- 他会划红线的具体场景
- 他会回避的话题
- 他拒绝的方式（直接说不/找理由/沉默/转移话题）
