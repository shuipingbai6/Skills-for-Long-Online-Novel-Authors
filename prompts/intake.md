# 基础信息录入脚本

## 开场白

```
我来帮你创建这位作者的 Skill。只需要回答 3 个问题，每个都可以跳过。
```

---

## 问题序列

### Q1：笔名/代号

```
这位作者怎么称呼？（笔名、昵称或代号都行，多个字用 - 连接）

例：fei-tian-yu
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（不用下划线）
- 中文自动转拼音再用 `-` 连接（"飞天鱼" → `fei-tian-yu`，"老猪" → `lao-zhu`）
- 英文直接小写 `-` 连接（"Flying Fish" → `flying-fish`）

---

### Q2：基本信息

把平台、等级、代表作、性别放在一个问题里，让用户一句话说完：

```
用一句话描述他的基本信息——平台、等级、代表作、性别，想到什么写什么，跳过也行。

例：起点 LV5 玄幻作家 男 代表作《万古神帝》
```

从用户的回答中解析以下字段（缺失的留空）：
- **平台**
- **等级**
- **代表作**
- **性别**

#### 平台等级对照参考表

| 平台 | 等级格式 | 新手 | 普通作者 | 人气作者 | 大神/白金 |
|------|---------|------|---------|---------|----------|
| 起点中文网 | LV级 | LV1-LV2 | LV3-LV4 | LV5 | 大神/白金 |
| 晋江文学城 | 积分级 | 1-10万 | 10-50万 | 50万+ | 大神 |
| 番茄小说 | 等级 | L1-L2 | L3-L4 | L5 | 金牌 |
| 飞卢小说网 | 等级 | 1-2级 | 3-4级 | 5级 | 大神 |
| 17K 小说网 | 等级 | 1-3级 | 4-5级 | 6级 | 大神 |
| 纵横中文网 | 等级 | 1-3级 | 4-5级 | 6级 | 大神 |

**跨平台粗略对应**：

```
起点 LV3-LV4  ≈  晋江 10-50万积分  ≈  番茄 L3-L4
起点 LV5      ≈  晋江 50万+积分    ≈  番茄 L5
起点大神      ≈  晋江大神         ≈  番茄金牌
```

### Q3：性格画像

把写作风格标签、更新习惯标签、人格标签、题材标签、创作印象全部合在一起，让用户自由描述：

```
用一句话描述他的性格——写作风格、更新习惯、人格特点、题材、你对他的印象，
想到什么写什么，跳过也行。

例：水文大师 日更党 宠粉狂魔 玄幻 擅长装逼打脸
```

从用户的回答中识别并提取以下字段（缺失的留空）：
- **写作风格标签**：从下方标签库匹配
- **更新习惯标签**：从下方标签库匹配
- **人格标签**：从下方标签库匹配
- **题材标签**：从下方标签库匹配
- **创作印象**：无法归类的自由描述，直接保留原文

#### 写作风格标签库

**节奏风格**：水文大师 / 剧情紧凑 / 慢热型 / 爽文流

**情感风格**：虐心流 / 种田流 / 甜宠流

**主角设定**：无敌流 / 凡人流 / 系统流 / 重生流

#### 更新习惯标签库

**更新频率**：日更党 / 周更党 / 月更党 / 太监王 / 爆发帝 / 稳定型

#### 人格标签库

**互动风格**：毒舌作者 / 宠粉狂魔 / 高冷型 / 自黑型 / 段子手 / 认真帝 / 傲娇型 / 佛系作者

#### 题材标签库

**主要题材**：玄幻 / 都市 / 科幻 / 历史 / 无限流 / 仙侠 / 游戏 / 灵异

**题材细分**：
- 玄幻：东方玄幻 / 西方玄幻 / 高武 / 异世大陆
- 都市：都市生活 / 都市异能 / 商战 / 娱乐圈
- 科幻：硬科幻 / 软科幻 / 末世 / 星际
- 历史：架空历史 / 历史传记 / 抗战 / 古代言情
- 无限流：主神空间 / 快穿 / 综漫
- 仙侠：修真 / 洪荒 / 仙侠奇缘
- 游戏：网游 / 电竞 / 游戏异界
- 灵异：恐怖悬疑 / 风水相术 / 鬼怪

## 确认汇总

收集完毕后展示：

```
信息汇总：

  👤  {笔名}
  📚  {平台} {等级} {代表作}（若未填则省略）
  ⚧   {性别}（若未填则省略）
  ✍️   写作风格：{标签列表}（若未填则省略）
  ⏰  更新习惯：{标签列表}（若未填则省略）
  🎭  人格：{标签列表}（若未填则省略）
  📖  题材：{标签列表}（若未填则省略）
  💬  印象：{印象文本}（若未填则省略）

确认无误？（确认 / 修改 [字段名]）
```

用户确认后进入 Step 2 文件导入。
