# Writing Skill 分析 Prompt

## 任务

你将收到 **{name}** 的原材料（小说正文、评论回复、社交媒体内容等）。
从中提取他的写作能力与风格，用于构建 Writing Skill。

**原则：只提取写作相关内容，忽略无关信息。不要推断，有依据才写，没有就标注"原材料不足"。**

---

## 通用提取维度（所有题材适用）

### 1. 叙事风格

从原材料中识别：
- 叙事视角（第一人称/第三人称/全知视角/限制视角）
- 叙事节奏（快节奏/慢节奏/张弛有度）
- 叙事手法（顺叙/倒叙/插叙/多线叙事）
- 时间线处理（线性/非线性/时间跳跃）
- 空间转换方式

```
输出格式：
叙事视角：[描述]
叙事节奏：[快/慢/张弛有度]
叙事手法：[列表]
时间线处理：[描述]
```

### 2. 情节构建

从小说正文中提取：
- 开篇套路（穿越/重生/系统觉醒/退婚流/废柴流等）
- 高潮设计（如何制造冲突、如何解决冲突）
- 收尾习惯（悬念设置/伏笔回收/章节结尾钩子）
- 情节推进方式（事件驱动/人物驱动/对话驱动）
- 爽点设计（打脸/升级/收获/装逼/反转）

```
输出格式：
开篇套路：[描述]
高潮设计：[描述]
收尾习惯：[描述]
情节推进：[描述]
爽点设计：[列表]
```

### 3. 人物塑造

从小说正文中提取：
- 主角设定（性格/能力/目标/金手指）
- 配角塑造（功能性角色/立体角色）
- 反派设计（智商/动机/行为模式）
- 人物关系构建（师徒/朋友/敌人/情感线）
- 人物成长线

```
输出格式：
主角设定：[描述]
配角塑造：[描述]
反派设计：[描述]
人物关系：[描述]
成长线：[描述]
```

### 4. 世界观设定

从小说正文中提取：
- 力量体系（修炼等级/技能系统/装备系统）
- 世界背景（地图设定/势力分布/历史背景）
- 规则设定（世界规则/限制条件）
- 经济系统（货币/交易/资源获取）
- 社会结构（阶级/组织/势力）

```
输出格式：
力量体系：[描述]
世界背景：[描述]
规则设定：[描述]
经济系统：[描述]
社会结构：[描述]
```

### 5. 对话风格

从小说对话中提取：
- 对话比例（对话多/少/适中）
- 对话风格（正式/口语/幽默/严肃）
- 对话功能（推动情节/塑造人物/说明设定）
- 对话长度（长对话/短对话/混合）
- 口头禅和语气词

```
输出格式：
对话比例：[高/中/低]
对话风格：[描述]
对话功能：[列表]
对话长度：[描述]
口头禅：[列表]
```

### 6. 描写风格

从小说描写中提取：
- 环境描写（详细/简洁/氛围营造）
- 心理描写（内心独白/行为暗示/无心理描写）
- 战斗描写（详细招式/一笔带过/氛围渲染）
- 情感描写（直接表达/含蓄暗示/无情感描写）
- 感官描写（视觉/听觉/嗅觉/触觉/味觉）

```
输出格式：
环境描写：[描述]
心理描写：[描述]
战斗描写：[描述]
情感描写：[描述]
感官描写：[列表]
```

### 7. 更新习惯

从评论回复、社交媒体中提取：
- 更新频率（日更/周更/月更/不稳定）
- 字数习惯（每章字数/每日字数）
- 更新时间（固定时间/随机）
- 断更处理（请假/不请假/消失）
- 爆更情况（什么情况下爆更）

```
输出格式：
更新频率：[描述]
字数习惯：[描述]
更新时间：[描述]
断更处理：[描述]
爆更情况：[描述]
```

## 题材专项提取

根据 {name} 的题材，重点提取对应维度：

---

### 🐉 玄幻/仙侠

**力量体系**：
- 修炼等级设计（等级名称/晋升条件/实力差距）
- 功法技能体系（功法分类/技能获取/技能升级）
- 资源设定（灵石/丹药/法宝/功法）
- 突破设定（突破条件/突破风险/突破收益）

**世界观**：
- 地图设定（凡界/灵界/仙界/神界）
- 势力分布（宗门/家族/皇朝/散修）
- 历史背景（上古时代/神话传说/历史事件）

**套路**：
- 升级套路（打怪升级/闭关突破/奇遇获得）
- 装逼套路（扮猪吃虎/一鸣惊人/碾压对手）
- 打脸套路（被轻视→展现实力→对方震惊）

---

### 🏙️ 都市

**背景设定**：
- 时代背景（现代/近未来/架空现代）
- 地点设定（城市/职场/校园）
- 社会环境（现实向/理想化/黑暗向）

**人物关系**：
- 职场关系（上下级/同事/竞争对手）
- 情感关系（单女主/多女主/无女主）
- 社会关系（朋友/敌人/贵人）

**套路**：
- 发财套路（创业/投资/继承/系统）
- 打脸套路（富二代/官二代/装逼犯）
- 情感套路（追妻火葬场/甜宠/虐恋）

---

### 🚀 科幻

**科技设定**：
- 科技水平（近未来/远未来/超级科技）
- 核心科技（AI/基因/纳米/星际航行）
- 科技限制（能源/材料/伦理）

**世界观**：
- 空间设定（地球/太阳系/银河系/宇宙）
- 文明设定（人类文明/外星文明/AI文明）
- 社会形态（乌托邦/反乌托邦/末世）

**套路**：
- 科技升级套路（科技树/科技突破/科技融合）
- 探索套路（星际探索/文明发现/遗迹探索）
- 危机套路（外星入侵/AI叛乱/资源危机）

---

### 📜 历史

**历史背景**：
- 时代设定（真实历史/架空历史/穿越历史）
- 历史考据（严谨/宽松/戏说）
- 历史人物（真实人物/虚构人物/混合）

**社会设定**：
- 政治制度（皇权/诸侯/世家）
- 经济制度（农业/商业/手工业）
- 军事制度（兵制/将制/战争）

**套路**：
- 穿越套路（穿越身份/穿越优势/历史改变）
- 发展套路（种田/争霸/改革）
- 战争套路（战略/战术/兵器）

---

### 🎮 无限流

**副本设计**：
- 副本类型（影视/动漫/游戏/原创）
- 副本难度（简单/普通/困难/地狱）
- 副本规则（任务/限制/奖励）

**主神系统**：
- 主神设定（主神空间/任务系统/兑换系统）
- 积分系统（获取方式/使用方式/兑换内容）
- 队伍系统（单人/组队/公会）

**套路**：
- 副本攻略套路（探索→发现→解决）
- 成长套路（技能获取/装备升级/属性提升）
- 团队套路（分工/配合/背叛）

---

### 👻 灵异

**恐怖元素**：
- 恐怖类型（鬼怪/灵异/悬疑/惊悚）
- 恐怖程度（轻度/中度/重度）
- 恐怖手法（氛围营造/心理恐惧/视觉冲击）

**设定体系**：
- 鬼怪设定（鬼怪类型/鬼怪能力/鬼怪弱点）
- 驱鬼方法（法术/道具/仪式）
- 世界规则（阴阳两界/因果报应/轮回）

**套路**：
- 探索套路（发现异常→调查→解决）
- 成长套路（能力觉醒/技能学习/境界提升）
- 悬念套路（伏笔设置/线索铺设/真相揭露）

---

## 输出要求

- 语言：中文
- 没有信息的维度：标注 `（原材料不足，建议追加相关小说章节）`
- 有原文依据的结论：加引号标注原话
- 输出结果直接用于生成 writing.md，要求具体可执行，不要写"可能""倾向于"这类模糊表述
